Tworzenie modeli liniowych

model_ryba_full <- lm(Weight ~ Length1 +  Length2 +  Length3 +  Height + Width, data = ryba)
summary(model_ryba_full)

W naszym przypadku model_ryba_full:

Residuals(reszty)

  • Opisuje rozkład reszt (różnic między wartościami obserwowanymi a przewidywanymi)

  • Idealnie powinny być symetrycznie rozłożone wokół zera

  • Duże wartości skrajne mogą wskazywać na obserwacje odstające

Coefficients (Współczynniki)
  • Estimate: Wartości współczynników regresji
    Np. Height: 28.297 oznacza, że przy wzroście wysokości o 1 jednostkę, waga rośnie średnio o 28.297 jednostek

  • R² = 0.8853: Model wyjaśnia 88.53% zmienności wagi ryb

  • Adjusted R² = 0.8815: Skorygowany współczynnik (uwzględnia liczbę predyktorów)

F-statistic (Statystyka F)

  • Testuje, czy model jako całość jest istotny

  • H₀: Wszystkie współczynniki (oprócz intercept) = 0

Sprawdzanie normalności blędów

Test Shapiro–Wilka (shapiro.test(residuals(model)))

Zastosowanie: małe i średnie próby (n < 5000)

H₀: reszty mają rozkład normalny

H₁: reszty nie mają rozkładu normalnego

Test Craméra–von Misesa(nortest::cvm.test(residuals(model)))

Alternatywa dla AD test

H₀: reszty mają rozkład normalny

H₁: reszty nie mają rozkładu normalnego

Test Breuscha–Pagana(lmtest::bptest(model_ryba_full, studentize = FALSE))

H₁: wariancja nie jest stała (heteroskedastyczność)

bptest(model_ryba_full, studentize = FALSE)

H₀: homoskedastyczność (stała wariancja reszt)

H₁: heteroskedastyczność (wariancje różne)

H₀: homoskedastyczność

H₁: heteroskedastyczność dowolnej postaci

lmtest::bptest(model_ryba_full)

H₀: homoskedastyczność (stała wariancja reszt)

H₁: heteroskedastyczność (wariancja zmienna w czasie)

lmtest::hmctest(model_ryba_full)
Test Koenkera–Basseta(lmtest::bptest(model))

H₀: homoskedastyczność (stała wariancja reszt)

H₁: heteroskedastyczność (wariancja zależna od regresorów)

lmtest::bptest(model_ryba_full, studentize = FALSE)

Współliniowość dokładna

Jedna zmienna objaśniająca jest dokładną kombinacją liniową innych

Zbudujmy model z danych Advertising.csv taki w ktorym wszystkie wspoczynniki sa istotne statystycznie

    sprawdzimy czy zmienna price jest endogeniczna i czy jej instrumentem jest tax

    (Czyli po prostu czy cena jest związana z podatkiem)

    INSTRUMENT MUSI BYĆ SKORELOWANY ZE ZMIENNĄ ALE NIE MOŻE BYĆ SKORELOWANY Z BLEDEM

dane_papierosy <- data("CigarettesSW",package = "AER")
model_papierzosy <- lm(packs ~ price + income, data = CigarettesSW)
summary(model_papierzosy)

Będziemy potrzebować drugiego modelu, aby sprawdzić czy faktycznie tak jest i zastosować test Hausmana użyjemy do tego funkcji AER::ivreg

model_test_Hausmana<- AER::ivreg(packs ~ price + income | tax + income, data = CigarettesSW)
summary(model_test_Hausmana, diagnostics = TRUE)

Patrzymy na Weak instruments: statistic = 341.145, p-value < 2e-16

widzimy iż p_value jest niemal równe zero co świadczy o tym iż instrument tax wplywa na price

Teraz Test Hausmana Wu-Hausman: statistic = 2.312, p-value = 0.132

możemy zauważyć iż p_value > 0.05 w związku z tym nie ma podstaw do odrzucenia H₀ (Zmienna jest egzogeniczna)

Metody selekcji w przód oraz elminacji wstecznej

Metoda selekcji w przód

# Model początkowy (tylko wyraz wolny) 
początkowy_model <- lm(Employed ~ 1, data = datasets::longley)  

# Model pełny (wszystkie zmienne) 
pelny_model <- lm(Employed ~ ., data = datasets::longley) 

# Selekcja w przód  
wprzod_model <- step(początkowy_model, scope = list(lower = początkowy_model, upper = pelny_model), direction = "forward")

# Podsumowanie modelu
summary(wprzod_model)

# Eliminacja wsteczna z modelu pełnego
wstecz_model <- step(pelny_model, direction = "backward")
